# Facter Docs

Documentacao centralizada do ecossistema Facter — SaaS multi-tenant para gestao de frotas, manutencao e operacoes.

## Projetos

| Projeto | Status | Descricao |
|---------|--------|-----------|
| **Truck** | Producao | Gestao de frotas e manutencao |
| **Hub** | Desenvolvimento | SSO, billing e assinaturas |
| **Design System** | Producao | `@facter/ds-core` — componentes compartilhados |

## Stack

- **Backend**: NestJS + Prisma + PostgreSQL + Redis
- **Frontend**: React + Vite + TanStack Query + Zustand + Tailwind CSS
- **Design System**: `@facter/ds-core` (Radix UI + Tailwind)
